# Question 7

**Create a class named Book with data members bookId, title, and author.
a)​ Write a Java program to serialize a Book object into a file.
b)​ Deserialize the object from the file.
c)​ Display the book information on the console.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.*;

// The class must implement Serializable
class Book implements Serializable {
    int bookId;
    String title;
    String author;

    public Book(int bookId, String title, String author) {
        this.bookId = bookId;
        this.title = title;
        this.author = author;
    }

    public void display() {
        System.out.println("Book ID: " + bookId + ", Title: '" + title + "', Author: " + author);
    }
}

public class Main {
    public static void main(String[] args) {
        Book book = new Book(5001, "Advanced Java Programming", "John Doe");
        String filename = "book.ser";

        // a) Serialize the Book object
        try (ObjectOutputStream out = new ObjectOutputStream(new FileOutputStream(filename))) {
            out.writeObject(book);
            System.out.println("Book object serialized to " + filename);
        } catch (IOException e) {
            e.printStackTrace();
        }

        Book deserializedBook = null;

        // b) Deserialize the object
        try (ObjectInputStream in = new ObjectInputStream(new FileInputStream(filename))) {
            deserializedBook = (Book) in.readObject();
            System.out.println("Book object deserialized from " + filename);
        } catch (IOException | ClassNotFoundException e) {
            e.printStackTrace();
        }

        // c) Display information
        if (deserializedBook != null) {
            System.out.println("\n--- Deserialized Book Details ---");
            deserializedBook.display();
        }
    }
}
```

**Sample Output:**
```text
Book object serialized to book.ser
Book object deserialized from book.ser

--- Deserialized Book Details ---
Book ID: 5001, Title: 'Advanced Java Programming', Author: John Doe
```
