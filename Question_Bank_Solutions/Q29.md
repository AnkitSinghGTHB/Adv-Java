### 1. The Java Event Delegation Model
The Event Delegation Model is the standard architecture used in Java to handle user interactions within a GUI. In this model, the responsibility for processing an event is completely "delegated" from the component that fired it over to a specialized "Listener" object. This creates clean separation between the visual GUI code and the logical event-handling code.

### 2. Roles in the Event Delegation Model
The model revolves around three key participants:
1.  **Event Source (Component):** This is the actual GUI component that the user interacts with (e.g., a `JButton`, `JTextField`). It is the source that generates the event and delegates it to registered listeners.
2.  **Event Object:** When the source is triggered, it creates an Event Object. This object encapsulates all the contextual information about what just happened (e.g., coordinates of the mouse click, timestamp, key pressed).
3.  **Event Listener:** This is an interface implemented by a separate class. It silently registers itself to the Event Source and waits. Once the event occurs, the source invokes the specific method residing inside the Listener, passing the Event Object over for processing.

### 3. Example Illustration
```java
import javax.swing.*;
import java.awt.event.*;

public class EventDelegationDemo {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Event Delegation Demo");
        
        // 1. The Event Source
        JButton btnClick = new JButton("Click Me");
        
        // 3. The Event Listener (Anonymous Inner Class implementing ActionListener)
        btnClick.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                // 2. The 'e' parameter is the generated Event Object
                long timestamp = e.getWhen(); 
                System.out.println("Button was clicked! Action Command: " + e.getActionCommand());
                System.out.println("Timestamp: " + timestamp);
            }
        });
        
        frame.add(btnClick);
        frame.setSize(300, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}
```

**Sample Output:**
```text
[A GUI Window titled "Event Delegation Demo" appears.]
Button was clicked! Action Command: Click Me
Timestamp: 1678886300000
```
