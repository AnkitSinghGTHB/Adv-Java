# Question 19

**Write a Java program to create a JavaBean named AttendanceBean that stores attendance
status (Present/Absent) for multiple days using an indexed property.
●​ Provide methods to set and get attendance status by index.
●​ Demonstrate assigning attendance for at least 10 days.
●​ Display the attendance record in the main() method.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.Serializable;

public class AttendanceBean implements Serializable {
    public AttendanceBean() {
        attendanceRecords = new boolean[10]; // Requires at least 10 days
    }
    
    private boolean[] attendanceRecords;

    // Indexed Setter
    public void setAttendanceRecords(int index, boolean isPresent) {
        if (index >= 0 && index < attendanceRecords.length) {
            this.attendanceRecords[index] = isPresent;
        }
    }

    // Indexed Getter
    public boolean getAttendanceRecords(int index) {
        if (index >= 0 && index < attendanceRecords.length) {
            return this.attendanceRecords[index];
        }
        return false; 
    }

    public void setAttendanceRecords(boolean[] records) {
        this.attendanceRecords = records;
    }

    public boolean[] getAttendanceRecords() {
        return this.attendanceRecords;
    }

    public static void main(String[] args) {
        AttendanceBean attendanceBean = new AttendanceBean();

        // Demonstrate assigning attendance for at least 10 days
        attendanceBean.setAttendanceRecords(0, true);  // Day 1: Present
        attendanceBean.setAttendanceRecords(1, true);  // Day 2: Present
        attendanceBean.setAttendanceRecords(2, false); // Day 3: Absent
        attendanceBean.setAttendanceRecords(3, true);  // Day 4: Present
        attendanceBean.setAttendanceRecords(4, true);  // Day 5: Present
        attendanceBean.setAttendanceRecords(5, true);  // Day 6: Present
        attendanceBean.setAttendanceRecords(6, false); // Day 7: Absent
        attendanceBean.setAttendanceRecords(7, false); // Day 8: Absent
        attendanceBean.setAttendanceRecords(8, true);  // Day 9: Present
        attendanceBean.setAttendanceRecords(9, true);  // Day 10: Present

        // Display the attendance record
        System.out.println("--- 10-Day Student Attendance Record ---");
        for (int i = 0; i < 10; i++) {
            String status = attendanceBean.getAttendanceRecords(i) ? "Present" : "Absent";
            System.out.println("Day " + (i + 1) + ": " + status);
        }
    }
}
```

```text
Day 4: Present
Day 5: Present
Day 6: Present
Day 7: Absent
Day 8: Absent
Day 9: Present
Day 10: Present
```
