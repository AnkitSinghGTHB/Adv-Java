# Question 6

**Create a class named Employee with data members empId, empName, and salary.
a)​ Serialize an object of the Employee class and store it in a file.
b)​ Then deserialize the object from the file.
c)​ Display the employee details after deserialization.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.*;

// The class must implement Serializable
class Employee implements Serializable {
    private static final long serialVersionUID = 1L; // Recommended best practice
    int empId;
    String empName;
    double salary;

    public Employee(int empId, String empName, double salary) {
        this.empId = empId;
        this.empName = empName;
        this.salary = salary;
    }

    public void display() {
        System.out.println("Employee ID: " + empId + ", Name: " + empName + ", Salary: $" + salary);
    }
}

public class Main {
    public static void main(String[] args) {
        Employee emp = new Employee(101, "Alice Smith", 85000.0);
        String filename = "employee.ser";

        // a) Serialize the Employee object
        ObjectOutputStream out = null;
        try {
            out = new ObjectOutputStream(new FileOutputStream(filename));
            out.writeObject(emp);
            System.out.println("Employee object serialized successfully.");
        } catch (IOException e) {
            e.printStackTrace();
        } finally {
            try {
                if (out != null) out.close();
            } catch (IOException ex) {
                ex.printStackTrace();
            }
        }

        Employee deserializedEmp = null;

        // b) Deserialize the object
        ObjectInputStream in = null;
        try {
            in = new ObjectInputStream(new FileInputStream(filename));
            deserializedEmp = (Employee) in.readObject();
            System.out.println("Employee object deserialized successfully.");
        } catch (IOException | ClassNotFoundException e) {
            e.printStackTrace();
        } finally {
            try {
                if (in != null) in.close();
            } catch (IOException ex) {
                ex.printStackTrace();
            }
        }

        // c) Display details
        if (deserializedEmp != null) {
            System.out.println("\n--- Displaying Deserialized Data ---");
            deserializedEmp.display();
        }
    }
}
```

**Sample Output:**
```text
Employee object serialized successfully.
Employee object deserialized successfully.

--- Displaying Deserialized Data ---
Employee ID: 101, Name: Alice Smith, Salary: $85000.0
```
