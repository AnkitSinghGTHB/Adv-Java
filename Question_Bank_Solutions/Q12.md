# Question 12

**Write a Java program to create a GUI application with:
a)​ One button labeled "Change Color"
b)​ A panel or frame
When the button is clicked, change the background color of the window using proper event
handling.**

---

### Solution

// TODO: Write solution here.

```java
import javax.swing.*;
import java.awt.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;
import java.util.Random;

public class ColorChangeApp extends JFrame {
    JButton btnChangeColor;
    JPanel panel;

    public ColorChangeApp() {
        setTitle("Color Changer");
        setSize(400, 300);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        // b) A panel to change its background color
        panel = new JPanel();
        panel.setLayout(new GridBagLayout()); // Center the button
        
        // a) One button labeled "Change Color"
        btnChangeColor = new JButton("Change Color");
        panel.add(btnChangeColor);
        add(panel);

        // Event handling to change color
        btnChangeColor.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                // Generate a random color
                Random rand = new Random();
                Color randomColor = new Color(rand.nextFloat(), rand.nextFloat(), rand.nextFloat());
                
                // Change the background color of the panel
                panel.setBackground(randomColor);
            }
        });
    }

    public static void main(String[] args) {
        new ColorChangeApp().setVisible(true);
    }
}
```
