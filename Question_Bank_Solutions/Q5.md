# Question 5

**Write a Java program to create a user-defined exception named
InvalidPriceException.​
The program should read the product price from the user.
a)​ If the price entered is less than or equal to zero, throw the custom exception.
b)​ Handle the exception properly with a meaningful error message.
c)​ Ensure the program continues execution normally after handling the exception.**

---

### Solution

// TODO: Write solution here.

```java
import java.util.Scanner;

// User-defined exception
class InvalidPriceException extends Exception {
    public InvalidPriceException(String message) {
        super(message);
    }
}

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter the product price: $");
        double price = scanner.nextDouble();
        
        try {
            // a) If price <= 0, throw the custom exception
            if (price <= 0) {
                throw new InvalidPriceException("A product's price cannot be zero or negative.");
            }
            System.out.println("Product price set to: $" + price);
            
        } catch (InvalidPriceException e) {
            // b) Handle the exception properly with a meaningful error message
            System.out.println("System Error: " + e.getMessage());
        } finally {
            scanner.close();
        }
        
        // c) Ensure the program continues execution normally after handling the exception
        System.out.println("Proceeding to inventory management module...");
    }
}
```
