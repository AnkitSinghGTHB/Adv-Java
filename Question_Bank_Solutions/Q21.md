# Question 21

**Write a Java program using JDBC to connect to a relational database and update the salary
of an employee in the Employee table.
●​ Accept empId and new salary from the user.
●​ Use PreparedStatement to perform the update operation.
●​ Display a message indicating whether the update was successful.
●​ Handle exceptions and close all resources properly**

---

### Solution

// TODO: Write solution here.

```java
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;
import java.sql.SQLException;
import java.util.Scanner;

public class UpdateEmployeeSalary {
    // Database credentials
    static final String DB_URL = "jdbc:mysql://localhost:3306/companyDb";
    static final String USER = "root";
    static final String PASS = "password";

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter Employee ID to update: ");
        int empId = scanner.nextInt();
        
        System.out.print("Enter New Salary: $");
        double newSalary = scanner.nextDouble();

        String sql = "UPDATE Employee SET salary = ? WHERE empId = ?";

        // Establish connection and prepared statement
        try (Connection conn = DriverManager.getConnection(DB_URL, USER, PASS);
             PreparedStatement pstmt = conn.prepareStatement(sql)) {

            // Set the dynamic parameters
            pstmt.setDouble(1, newSalary);
            pstmt.setInt(2, empId);

            int rowsUpdated = pstmt.executeUpdate();
            
            // Display indication of success
            if (rowsUpdated > 0) {
                System.out.println("Success: Employee salary updated to $" + newSalary);
            } else {
                System.out.println("Warning: No employee found with ID " + empId);
            }

        } catch (SQLException e) {
            System.err.println("SQL Exception: " + e.getMessage());
            e.printStackTrace();
        } finally {
            scanner.close();
            System.out.println("Cleanup completed.");
        }
    }
}
```
