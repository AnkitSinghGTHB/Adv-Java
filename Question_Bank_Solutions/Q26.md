# Question 26

**Define class and object in Java.​
Write a Java program to create a class Student with data members and methods. Create
objects and demonstrate method invocation.**

### 1. Definition: Class and Object
*   **Class:** A logical blueprint or template from which objects are created. It defines the structure (data members/fields) and behaviors (methods) that its objects will possess. A class does not consume memory when defined.
*   **Object:** A physical, real-world entity that is an instance of a class. When an object is instantiated, memory is allocated on the heap. Objects contain actual values for the class's data members.

### 2. Demonstration Program
The following program creates a `Student` class and demonstrates the invocation of its methods using an object instance.

```java
// 1. Define the Class (Blueprint)
class Student {
    // Data Members (State)
    int rollNo;
    String name;
    double grade;

    // Methods (Behavior)
    // Method to assign data
    public void setDetails(int r, String n, double g) {
        rollNo = r;
        name = n;
        grade = g;
    }

    // Method to display data
    public void displayProfile() {
        System.out.println("Roll Number: " + rollNo);
        System.out.println("Name: " + name);
        System.out.println("Grade GPA: " + grade);
        System.out.println("-------------------------");
    }
}

public class Main {
    public static void main(String[] args) {
        // 2. Create Objects (Instances of the class)
        Student s1 = new Student();
        Student s2 = new Student();

        // 3. Demonstrate Method Invocation
        s1.setDetails(101, "Alice Smith", 3.8);
        s2.setDetails(102, "Bob Johnson", 3.5);

        System.out.println("--- Student Database ---");
        // Invoking display method
        s1.displayProfile();
        s2.displayProfile();
    }
}
```
