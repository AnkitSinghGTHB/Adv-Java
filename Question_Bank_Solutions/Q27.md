# Question 27

**What is an interface in Java?​
Explain different access modifiers (private, default, protected, public) with examples.​
Demonstrate how a class implements an interface.**

### 1. Interface in Java
An interface is a pure abstract blueprint of a class. It can contain only static constants and abstract methods (prior to Java 8). It is used to achieve 100% abstraction and to support multiple inheritance of properties. Classes use the `implements` keyword to inherit an interface, and they must provide concrete definitions for all of its abstract methods.

### 2. Access Modifiers in Java
Access modifiers determine the visibility and accessibility scope of classes, variables, and methods.
1.  **`private`**: The member is accessible *only* within the same class it is declared.
    *   *Example:* `private int bankBalance;` (Cannot be seen by other classes).
2.  **`default` (Package-Private)**: If no modifier is specified, it is accessible only within classes in the same package.
    *   *Example:* `int age;`
3.  **`protected`**: Accessible within the same package, and outside the package *only* through child subclasses (Inheritance).
    *   *Example:* `protected void calculateTax() { ... }`
4.  **`public`**: The member is accessible from anywhere across the entire application.
    *   *Example:* `public void displayResult() { ... }`

### 3. Demonstration Program
```java
// Define an Interface
interface Payable {
    // Implicitly public, static, and final
    double BONUS_RATE = 0.05; 
    
    // Implicitly public and abstract
    void calculateSalary(); 
}

// Class implementing the interface
class Employee implements Payable {
    private String name; // private modifier
    public double basePay; // public modifier

    public Employee(String name, double basePay) {
        this.name = name;
        this.basePay = basePay;
    }

    // Must provide implementation for the interface method
    @Override
    public void calculateSalary() {
        double total = basePay + (basePay * BONUS_RATE);
        System.out.println("Total Salary for " + name + " is: $" + total);
    }
}

public class Main {
    public static void main(String[] args) {
        Payable emp = new Employee("John", 50000);
        emp.calculateSalary();
    }
}
```
