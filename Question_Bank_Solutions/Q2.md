# Question 2

**Write a Java program to create a user-defined exception named
InvalidMarksException.​
The program should read marks from the user.
a)​ If the marks entered are less than 0 or greater than 100, throw the custom exception.
b)​ Handle the exception and display a suitable error message.
c)​ Allow the program to continue execution normally.**

---

### Solution

// TODO: Write solution here.

```java
import java.util.Scanner;

// Custom exception
class InvalidMarksException extends Exception {
    public InvalidMarksException(String message) {
        super(message);
    }
}

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter the student's marks (0-100): ");
        int marks = scanner.nextInt();
        
        try {
            // a) If marks are < 0 or > 100, throw exception
            if (marks < 0 || marks > 100) {
                throw new InvalidMarksException("Invalid marks entered! Marks must be between 0 and 100.");
            }
            System.out.println("Marks recorded successfully: " + marks);
            
        } catch (InvalidMarksException e) {
            // b) Handle the exception and print the error message
            System.out.println("Error: " + e.getMessage());
        } finally {
            scanner.close();
        }
        
        // c) Allow the program to continue execution normally
        System.out.println("Execution continues naturally beyond the try-catch block.");
    }
}
```
