# Question 9

**Create a class named Product with data members productId, productName, and
price.
a)​ Serialize a Product object and store it in a file named product.dat.
b)​ Deserialize the object from the file.
c)​ Print the product details.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.*;

// The class must implement Serializable
class Product implements Serializable {
    int productId;
    String productName;
    double price;

    public Product(int productId, String productName, double price) {
        this.productId = productId;
        this.productName = productName;
        this.price = price;
    }

    public void display() {
        System.out.println("Product ID: " + productId + " | Name: " + productName + " | Price: $" + price);
    }
}

public class Main {
    public static void main(String[] args) {
        Product product = new Product(101, "Wireless Mouse", 25.99);
        String filename = "product.dat"; // Specific filename requested

        // a) Serialize the Product object into product.dat
        try (ObjectOutputStream out = new ObjectOutputStream(new FileOutputStream(filename))) {
            out.writeObject(product);
            System.out.println("Product object serialized into " + filename);
        } catch (IOException e) {
            e.printStackTrace();
        }

        Product deserializedProduct = null;

        // b) Deserialize the object from the file
        try (ObjectInputStream in = new ObjectInputStream(new FileInputStream(filename))) {
            deserializedProduct = (Product) in.readObject();
            System.out.println("Product object deserialized from " + filename);
        } catch (IOException | ClassNotFoundException e) {
            e.printStackTrace();
        }

        // c) Print the product details
        if (deserializedProduct != null) {
            System.out.println("\n--- Extracted Product Details ---");
            deserializedProduct.display();
        }
    }
}
```
