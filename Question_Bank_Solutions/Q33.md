# Question 33

**Develop a Java program that demonstrates the difference between BorderLayout and
FlowLayout by placing the same set of buttons in two separate panels within the same frame.**

### 1. BorderLayout vs FlowLayout
*   **FlowLayout:** The default layout for Panels. It arranges components in a directional flow, much like lines of text in a paragraph (left to right, top to bottom). Components retain their preferred sizes.
*   **BorderLayout:** The default layout for Frames. It divides the container into five regions: `NORTH`, `SOUTH`, `EAST`, `WEST`, and `CENTER`. Each region can contain only one component, and the center expands to consume all remaining space.

### 2. Demonstration Program
```java
import javax.swing.*;
import java.awt.*;

public class LayoutComparisonApp extends JFrame {
    public LayoutComparisonApp() {
        setTitle("BorderLayout vs FlowLayout");
        setSize(500, 300);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        
        // Use GridLayout(1,2) to evenly split the frame vertically
        setLayout(new GridLayout(1, 2));

        // --- Panel 1: BorderLayout ---
        JPanel borderPanel = new JPanel();
        borderPanel.setLayout(new BorderLayout());
        // Background color for visual distinction
        borderPanel.setBackground(Color.LIGHT_GRAY); 
        
        borderPanel.add(new JButton("North"), BorderLayout.NORTH);
        borderPanel.add(new JButton("South"), BorderLayout.SOUTH);
        borderPanel.add(new JButton("East"), BorderLayout.EAST);
        borderPanel.add(new JButton("West"), BorderLayout.WEST);
        borderPanel.add(new JButton("Center"), BorderLayout.CENTER);

        // --- Panel 2: FlowLayout ---
        JPanel flowPanel = new JPanel();
        flowPanel.setLayout(new FlowLayout()); // Default anyway
        flowPanel.setBackground(Color.DARK_GRAY);

        // Contains the exact same buttons as the BorderLayout
        flowPanel.add(new JButton("Button 1 (North)"));
        flowPanel.add(new JButton("Button 2 (South)"));
        flowPanel.add(new JButton("Button 3 (East)"));
        flowPanel.add(new JButton("Button 4 (West)"));
        flowPanel.add(new JButton("Button 5 (Center)"));

        // Add both distinctive panels to the main frame
        add(borderPanel);
        add(flowPanel);
    }

    public static void main(String[] args) {
        new LayoutComparisonApp().setVisible(true);
    }
}
```
