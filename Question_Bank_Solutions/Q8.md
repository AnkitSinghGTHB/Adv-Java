# Question 8

**Create a class named BankAccount with data members accountNumber,
accountHolderName, and balance.
a)​ Serialize the BankAccount object into a file.
b)​ Deserialize it from the file.
c)​ Display the account details after reading from the file.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.*;

// The class must implement Serializable
class BankAccount implements Serializable {
    int accountNumber;
    String accountHolderName;
    double balance;

    public BankAccount(int accountNumber, String accountHolderName, double balance) {
        this.accountNumber = accountNumber;
        this.accountHolderName = accountHolderName;
        this.balance = balance;
    }

    public void display() {
        System.out.println("A/C Number: " + accountNumber + "\nHolder: " + accountHolderName + "\nBalance: $" + balance);
    }
}

public class Main {
    public static void main(String[] args) {
        BankAccount account = new BankAccount(99887766, "Jane Doe", 15250.75);
        String filename = "bankaccount.ser";

        // a) Serialize the BankAccount object
        try (ObjectOutputStream out = new ObjectOutputStream(new FileOutputStream(filename))) {
            out.writeObject(account);
            System.out.println("BankAccount object serialized successfully.");
        } catch (IOException e) {
            e.printStackTrace();
        }

        BankAccount deserializedAccount = null;

        // b) Deserialize the object
        try (ObjectInputStream in = new ObjectInputStream(new FileInputStream(filename))) {
            deserializedAccount = (BankAccount) in.readObject();
            System.out.println("BankAccount object deserialized successfully.");
        } catch (IOException | ClassNotFoundException e) {
            e.printStackTrace();
        }

        // c) Display account details
        if (deserializedAccount != null) {
            System.out.println("\n--- Deserialized Account Status ---");
            deserializedAccount.display();
        }
    }
}
```
