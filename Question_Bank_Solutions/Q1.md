# Question 1

**Write a Java program to create a user-defined exception named
InsufficientBalanceException.​
The program should read the account balance and withdrawal amount from the user.
a)​ If the withdrawal amount is greater than the available balance, throw the custom
exception.
b)​ Handle the exception properly and display an appropriate message.
c)​ Ensure the program continues execution normally after handling the exception.**

---

### Solution

// TODO: Write solution here.

```java
import java.util.Scanner;

// Custom Exception Class
class InsufficientBalanceException extends Exception {
    public InsufficientBalanceException(String message) {
        super(message);
    }
}

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        
        System.out.print("Enter current account balance: ");
        double balance = scanner.nextDouble();
        
        System.out.print("Enter withdrawal amount: ");
        double withdrawAmount = scanner.nextDouble();
        
        try {
            // a) If withdrawal amount > balance, throw custom exception
            if (withdrawAmount > balance) {
                throw new InsufficientBalanceException("Error: Insufficient balance for this withdrawal.");
            }
            // If valid, deduct and show success
            balance -= withdrawAmount;
            System.out.println("Withdrawal successful! Remaining balance: " + balance);
            
        } catch (InsufficientBalanceException e) {
            // b) Handle exception and display message
            System.out.println(e.getMessage());
        } finally {
            scanner.close();
        }
        
        // c) Program continues normally
        System.out.println("Thank you for using our banking services.");
    }
}
```
