# Question 23

**Write a Java program using JDBC to connect to a relational database and retrieve details
of a specific customer from a table named Customer having fields customerId,
customerName, and city.
●​ Accept customerId as input.
●​ Display the customer details if found.
●​ Handle all exceptions properly.
●​ Ensure proper closing of Connection, Statement, and ResultSet.**

---

### Solution

// TODO: Write solution here.

```java
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;
import java.sql.ResultSet;
import java.sql.SQLException;
import java.util.Scanner;

public class RetrieveCustomer {
    static final String DB_URL = "jdbc:mysql://localhost:3306/salesDb";
    static final String USER = "root";
    static final String PASS = "password";

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter Customer ID to retrieve details: ");
        int customerId = scanner.nextInt();

        String sql = "SELECT customerId, customerName, city FROM Customer WHERE customerId = ?";

        // Ensure proper closing by using try-with-resources 
        try (Connection conn = DriverManager.getConnection(DB_URL, USER, PASS);
             PreparedStatement pstmt = conn.prepareStatement(sql)) {

            pstmt.setInt(1, customerId);

            // Fetch the ResultSet
            try (ResultSet rs = pstmt.executeQuery()) {
                
                // Display customer details if found
                if (rs.next()) {
                    System.out.println("\n--- Customer Record Found ---");
                    System.out.println("Customer ID: " + rs.getInt("customerId"));
                    System.out.println("Name: " + rs.getString("customerName"));
                    System.out.println("City: " + rs.getString("city"));
                } else {
                    System.out.println("Customer ID " + customerId + " does not exist in the database.");
                }
            } // ResultSet auto-closes here

        } catch (SQLException e) {
            System.err.println("Failed to retrieve data: " + e.getMessage());
            e.printStackTrace();
        } finally {
            scanner.close();
            System.out.println("Ensured closing of Connection, Statement, and ResultSet.");
        }
    }
}
```
