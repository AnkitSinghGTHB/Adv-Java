# Question 18

**Write a Java program to create a JavaBean named SalesBean that stores quarterly sales
figures using an indexed property.
●​ Include methods to set and retrieve sales for each quarter.
●​ Set values for all four quarters.
●​ Display the quarterly sales data in the main() method.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.Serializable;

public class SalesBean implements Serializable {
    public SalesBean() {
        quarterlySales = new double[4]; // 4 Quarters in a year
    }
    
    private double[] quarterlySales;

    // Indexed Setter
    public void setQuarterlySales(int index, double sales) {
        if (index >= 0 && index < 4) {
            this.quarterlySales[index] = sales;
        }
    }

    // Indexed Getter
    public double getQuarterlySales(int index) {
        if (index >= 0 && index < 4) {
            return this.quarterlySales[index];
        }
        return -1.0; 
    }

    public void setQuarterlySales(double[] sales) {
        this.quarterlySales = sales;
    }

    public double[] getQuarterlySales() {
        return this.quarterlySales;
    }

    public static void main(String[] args) {
        SalesBean salesBean = new SalesBean();

        // Set values for all four quarters
        salesBean.setQuarterlySales(0, 150000.00); // Q1
        salesBean.setQuarterlySales(1, 175500.50); // Q2
        salesBean.setQuarterlySales(2, 160000.00); // Q3
        salesBean.setQuarterlySales(3, 210000.75); // Q4 (Holiday season)

        // Display the quarterly sales data
        System.out.println("--- Annual Quarterly Sales Report ---");
        double totalSales = 0;
        for (int i = 0; i < 4; i++) {
            double qSales = salesBean.getQuarterlySales(i);
            System.out.println("Quarter " + (i + 1) + " Sales: $" + qSales);
            totalSales += qSales;
        }
        System.out.println("Total Annual Sales: $" + totalSales);
    }
}
```

**Sample Output:**
```text
--- Annual Quarterly Sales Report ---
Quarter 1 Sales: $150000.0
Quarter 2 Sales: $175500.5
Quarter 3 Sales: $160000.0
Quarter 4 Sales: $210000.75
Total Annual Sales: $695501.25
```
