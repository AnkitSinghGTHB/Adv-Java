# Question 15

**Write a Java program to create a JavaBean named EmployeeSalaryBean that stores
monthly salaries for 12 months using an indexed property.
●​ Include methods to set and get salary for a specific month using index.
●​ Demonstrate setting salaries for different months.
●​ Display all monthly salaries in the main() method.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.Serializable;

// 1. Must implement Serializable
public class EmployeeSalaryBean implements Serializable {
    // 2. No-arg constructor
    public EmployeeSalaryBean() {
        // Initialize the array for 12 months
        monthlySalaries = new double[12];
    }
    
    // 3. Private properties
    private double[] monthlySalaries;

    // --- Indexed Property Methods ---
    
    // Setter for specific index
    public void setMonthlySalaries(int index, double salary) {
        if (index >= 0 && index < 12) {
            this.monthlySalaries[index] = salary;
        } else {
            System.out.println("Invalid month index. Must be 0-11.");
        }
    }

    // Getter for specific index
    public double getMonthlySalaries(int index) {
        if (index >= 0 && index < 12) {
            return this.monthlySalaries[index];
        }
        return -1; // Error value
    }

    // --- Array Property Methods ---
    public void setMonthlySalaries(double[] salaries) {
        this.monthlySalaries = salaries;
    }

    public double[] getMonthlySalaries() {
        return this.monthlySalaries;
    }

    // Main method to demonstrate usage
    public static void main(String[] args) {
        EmployeeSalaryBean salaryBean = new EmployeeSalaryBean();

        // Demonstrate setting salaries for different months
        salaryBean.setMonthlySalaries(0, 5000.0); // January
        salaryBean.setMonthlySalaries(1, 5200.0); // February
        salaryBean.setMonthlySalaries(2, 5200.0); // March
        salaryBean.setMonthlySalaries(11, 6000.0); // December (with bonus)

        // Display all monthly salaries
        System.out.println("--- Employee Monthly Salaries ---");
        System.out.println("Jan Salary: $" + salaryBean.getMonthlySalaries(0));
        System.out.println("Feb Salary: $" + salaryBean.getMonthlySalaries(1));
        System.out.println("Mar Salary: $" + salaryBean.getMonthlySalaries(2));
        System.out.println("Dec Salary: $" + salaryBean.getMonthlySalaries(11));
    }
}
```

**Sample Output:**
```text
--- Employee Monthly Salaries ---
Jan Salary: $5000.0
Feb Salary: $5200.0
Mar Salary: $5200.0
Dec Salary: $6000.0
```
