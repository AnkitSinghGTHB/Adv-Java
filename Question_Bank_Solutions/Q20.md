# Question 20

**Write a Java program using JDBC to connect to a relational database and insert a new
record into a table named Student having fields rollNo, name, and marks.
●​ Accept input from the user.
●​ Use PreparedStatement for insertion.
●​ Handle SQL and connection exceptions properly.
●​ Ensure all database resources are properly closed**

---

### Solution

// TODO: Write solution here.

```java
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;
import java.sql.SQLException;
import java.util.Scanner;

public class InsertStudent {
    // Database credentials
    static final String DB_URL = "jdbc:mysql://localhost:3306/universityDb";
    static final String USER = "root";
    static final String PASS = "password";

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("Enter Student Roll No: ");
        int rollNo = scanner.nextInt();
        scanner.nextLine(); // consume newline
        
        System.out.println("Enter Student Name: ");
        String name = scanner.nextLine();
        
        System.out.println("Enter Student Marks: ");
        int marks = scanner.nextInt();

        String sql = "INSERT INTO Student (rollNo, name, marks) VALUES (?, ?, ?)";

        // Using standard try-catch-finally to manually close connections
        Connection conn = null;
        PreparedStatement pstmt = null;
        try {
            conn = DriverManager.getConnection(DB_URL, USER, PASS);
            pstmt = conn.prepareStatement(sql);

            // Bind values securely to prevent SQL injection
            pstmt.setInt(1, rollNo);
            pstmt.setString(2, name);
            pstmt.setInt(3, marks);

            int rowsAffected = pstmt.executeUpdate();
            
            if (rowsAffected > 0) {
                System.out.println("Record inserted successfully!");
            }

        } catch (SQLException e) {
            System.err.println("Database error occurred: " + e.getMessage());
            e.printStackTrace();
        } finally {
            try {
                if (pstmt != null) pstmt.close();
                if (conn != null) conn.close();
            } catch (SQLException ex) {
                ex.printStackTrace();
            }
            scanner.close();
            System.out.println("Resources closed naturally via finally block.");
        }
    }
}
```

**Sample Output:**
```text
Enter Student Roll No: 101
Enter Student Name: Alice
Enter Student Marks: 95
Record inserted successfully!
Resources closed naturally via finally block.
```
