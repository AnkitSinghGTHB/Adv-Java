# Question 17

**Write a Java program to create a JavaBean named TemperatureBean that stores daily
temperature readings for a week using an indexed property.
●​ Implement setter and getter methods using index.
●​ Assign temperature values for all seven days.
●​ Display the weekly temperature report in the main() method.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.Serializable;

public class TemperatureBean implements Serializable {
    public TemperatureBean() {
        dailyTemperatures = new double[7]; // 7 days of the week
    }
    
    private double[] dailyTemperatures;

    // Indexed Setter
    public void setDailyTemperatures(int index, double temp) {
        if (index >= 0 && index < 7) {
            this.dailyTemperatures[index] = temp;
        }
    }

    // Indexed Getter
    public double getDailyTemperatures(int index) {
        if (index >= 0 && index < 7) {
            return this.dailyTemperatures[index];
        }
        return -999.99; // Error
    }

    public void setDailyTemperatures(double[] temps) {
        this.dailyTemperatures = temps;
    }

    public double[] getDailyTemperatures() {
        return this.dailyTemperatures;
    }

    public static void main(String[] args) {
        TemperatureBean tempBean = new TemperatureBean();

        // Assign temperature values for all seven days
        tempBean.setDailyTemperatures(0, 22.5); // Monday
        tempBean.setDailyTemperatures(1, 24.0); // Tuesday
        tempBean.setDailyTemperatures(2, 21.8); // Wednesday
        tempBean.setDailyTemperatures(3, 23.1); // Thursday
        tempBean.setDailyTemperatures(4, 25.4); // Friday
        tempBean.setDailyTemperatures(5, 27.2); // Saturday
        tempBean.setDailyTemperatures(6, 26.5); // Sunday

        // Display the weekly temperature report
        String[] days = {"Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"};
        
        System.out.println("--- Weekly Temperature Report ---");
        for (int i = 0; i < 7; i++) {
            System.out.println(days[i] + ": " + tempBean.getDailyTemperatures(i) + "°C");
        }
    }
}
```

**Sample Output:**
```text
--- Weekly Temperature Report ---
Monday: 22.5°C
Tuesday: 24.0°C
Wednesday: 21.8°C
Thursday: 23.1°C
Friday: 25.4°C
Saturday: 27.2°C
Sunday: 26.5°C
```
