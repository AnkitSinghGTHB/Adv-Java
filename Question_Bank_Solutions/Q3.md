# Question 3

**Write a Java program to create a user-defined exception named
WeakPasswordException.​
The program should accept a password from the user.
a)​ If the password length is less than 8 characters, throw the custom exception.
b)​ Handle the exception properly and display a meaningful message.
c)​ Continue program execution after handling the exception.**

---

### Solution

// TODO: Write solution here.

```java
import java.util.Scanner;

// User-defined Exception
class WeakPasswordException extends Exception {
    public WeakPasswordException(String message) {
        super(message);
    }
}

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Create a new password: ");
        String password = scanner.nextLine();
        
        try {
            // a) If password length < 8, throw custom exception
            if (password.length() < 8) {
                throw new WeakPasswordException("Password is too weak. It must be at least 8 characters long.");
            }
            System.out.println("Strong password accepted.");
            
        } catch (WeakPasswordException e) {
            // b) Handle the exception properly and display a meaningful message
            System.out.println("Security Alert: " + e.getMessage());
        } finally {
            scanner.close();
        }
        
        // c) Continue program execution after handling the exception
        System.out.println("Moving to the next step of the registration process...");
    }
}
```
