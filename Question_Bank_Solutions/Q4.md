# Question 4

**Write a Java program to create a user-defined exception named
InvalidTemperatureException.​
The program should read temperature input from the user.
a)​ If the temperature entered is below -50°C or above 60°C, throw the custom exception.
b)​ Handle the exception and display an appropriate message.
c)​ Ensure the program continues execution normally.**

---

### Solution

// TODO: Write solution here.

```java
import java.util.Scanner;

// Custom Exception
class InvalidTemperatureException extends Exception {
    public InvalidTemperatureException(String message) {
        super(message);
    }
}

public class Main {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter the current temperature in Celsius: ");
        double temperature = scanner.nextDouble();
        
        try {
            // a) If temperature is below -50 or above 60, throw exception
            if (temperature < -50 || temperature > 60) {
                throw new InvalidTemperatureException("Warning: Unrealistic temperature reading detected (" + temperature + "°C).");
            }
            System.out.println("Temperature logged: " + temperature + "°C");
            
        } catch (InvalidTemperatureException e) {
            // b) Handle the exception and display appropriate message
            System.out.println("Input Error: " + e.getMessage());
        } finally {
            scanner.close();
        }
        
        // c) Ensure the program continues execution normally
        System.out.println("Resuming normal monitoring operations...");
    }
}
```

**Sample Output:**
```text
Enter the current temperature in Celsius: 100
Input Error: Warning: Unrealistic temperature reading detected (100.0°C).
Resuming normal monitoring operations...
```
