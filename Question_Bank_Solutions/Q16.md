# Question 16

**Write a Java program to create a JavaBean named StudentGradesBean that stores grades
for multiple subjects using an indexed property.
●​ Provide methods to set and get grade by subject index.
●​ Demonstrate assigning grades for at least five subjects.
●​ Display all grades in the main() method.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.Serializable;

public class StudentGradesBean implements Serializable {
    public StudentGradesBean() {
        grades = new String[5]; // Stores grades for 5 subjects
    }
    
    private String[] grades;

    // Indexed Setter
    public void setGrades(int index, String grade) {
        if (index >= 0 && index < grades.length) {
            this.grades[index] = grade;
        }
    }

    // Indexed Getter
    public String getGrades(int index) {
        if (index >= 0 && index < grades.length) {
            return this.grades[index];
        }
        return null; 
    }

    public void setGrades(String[] grades) {
        this.grades = grades;
    }

    public String[] getGrades() {
        return this.grades;
    }

    public static void main(String[] args) {
        StudentGradesBean gradesBean = new StudentGradesBean();

        // Assigning grades for at least five subjects
        gradesBean.setGrades(0, "A+"); // Math
        gradesBean.setGrades(1, "A");  // Science
        gradesBean.setGrades(2, "B+"); // History
        gradesBean.setGrades(3, "A-"); // English
        gradesBean.setGrades(4, "A");  // Computer Science

        // Display all grades
        System.out.println("--- Student Subject Grades ---");
        for (int i = 0; i < 5; i++) {
            System.out.println("Subject " + (i + 1) + " Grade: " + gradesBean.getGrades(i));
        }
    }
}
```

**Sample Output:**
```text
--- Student Subject Grades ---
Subject 1 Grade: A+
Subject 2 Grade: A
Subject 3 Grade: B+
Subject 4 Grade: A-
Subject 5 Grade: A
```
