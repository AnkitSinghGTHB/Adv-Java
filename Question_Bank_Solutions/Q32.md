### 1. Thread Life Cycle Diagram
```text
                 [ NEW ]
                    | (t.start())
                    v
  +-----------> [ RUNNABLE ] <------------------+
  |                 |                           |
  | (Yield /        | (Scheduler picks t)       | (timeout ends,
  |  Time Slice     v                           |  notify(),
  |  Ends)      [ RUNNING ]                     |  lock acquired)
  |                 |                           |
  +-----------------+                           |
                    |                           |
                    | (sleep(), wait(), join()) |
                    v                           |
            [ BLOCKED / WAITING ] --------------+
           
              [ RUNNING ]
                    | (run() completes)
                    v
              [ TERMINATED ]
```

### 2. InterruptedException Causes
An `InterruptedException` is thrown specifically when a thread is asleep or waiting, and another thread invokes `.interrupt()` on it to forcefully wake it up.
*   **`sleep(millis)`:** Puts the thread to sleep for exactly `millis` milliseconds. If interrupted during this sleep phase, it throws the exception immediately.
*   **`wait()`:** Causes the thread to wait indefinitely until another thread calls `notify()`. If interrupted while waiting on the object monitor, it throws.
*   **`join()`:** Causes the current thread to wait until the target thread finishes executing. If interrupted while stuck waiting for the other thread to finish, it throws.

### 3. Runnable Interface Demonstration
```java
// Implement the Runnable interface
class ActionThread implements Runnable {
    public void run() {
        for (int i = 1; i <= 7; i++) {
            System.out.println(i + ". Multithreading in Action");
            try {
                // 2-second delay in Timed Waiting state
                Thread.sleep(2000); 
            } catch (InterruptedException e) {
                System.out.println("Execution Interrupted.");
            }
        }
    }
}

public class Main {
    public static void main(String[] args) {
        // Pass the Runnable to the Thread constructor
        Thread actionT = new Thread(new ActionThread());
        actionT.start(); // Invokes run() under a new call stack
    }
}
```

**Sample Output:**
```text
1. Multithreading in Action
2. Multithreading in Action
...
7. Multithreading in Action
```
