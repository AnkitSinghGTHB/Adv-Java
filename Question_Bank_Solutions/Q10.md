# Question 10

**Create a class named Teacher with data members teacherId, teacherName, and
subject.
a)​ Serialize the Teacher object and save it into a file.
b)​ Deserialize the object from the file.
c)​ Display the teacher information after deserialization.**

---

### Solution

// TODO: Write solution here.

```java
import java.io.*;

// The class must implement Serializable
class Teacher implements Serializable {
    int teacherId;
    String teacherName;
    String subject;

    public Teacher(int teacherId, String teacherName, String subject) {
        this.teacherId = teacherId;
        this.teacherName = teacherName;
        this.subject = subject;
    }

    public void display() {
        System.out.println("Teacher ID: " + teacherId + "\nName: " + teacherName + "\nSubject: " + subject);
    }
}

public class Main {
    public static void main(String[] args) {
        Teacher teacher = new Teacher(401, "Dr. Alan Turing", "Computer Science");
        String filename = "teacher.ser";

        // a) Serialize the Teacher object
        try (ObjectOutputStream out = new ObjectOutputStream(new FileOutputStream(filename))) {
            out.writeObject(teacher);
            System.out.println("Teacher object serialized and saved.");
        } catch (IOException e) {
            e.printStackTrace();
        }

        Teacher deserializedTeacher = null;

        // b) Deserialize the object
        try (ObjectInputStream in = new ObjectInputStream(new FileInputStream(filename))) {
            deserializedTeacher = (Teacher) in.readObject();
            System.out.println("Teacher object deserialized completely.");
        } catch (IOException | ClassNotFoundException e) {
            e.printStackTrace();
        }

        // c) Display information
        if (deserializedTeacher != null) {
            System.out.println("\n--- Deserialized Teacher Profile ---");
            deserializedTeacher.display();
        }
    }
}
```

**Sample Output:**
```text
Teacher object serialized and saved.
Teacher object deserialized completely.

--- Deserialized Teacher Profile ---
Teacher ID: 401
Name: Dr. Alan Turing
Subject: Computer Science
```
