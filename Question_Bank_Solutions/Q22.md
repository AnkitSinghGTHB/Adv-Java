# Question 22

**Write a Java program using JDBC to connect to a relational database and delete a record
from a table named Product with fields productId, productName, and price.
●​ Accept productId from the user.
●​ Perform the delete operation using JDBC.
●​ Display appropriate messages.
●​ Handle exceptions and ensure proper resource cleanup.**

---

### Solution

// TODO: Write solution here.

```java
import java.sql.Connection;
import java.sql.DriverManager;
import java.sql.PreparedStatement;
import java.sql.SQLException;
import java.util.Scanner;

public class DeleteProduct {
    static final String DB_URL = "jdbc:mysql://localhost:3306/storeDb";
    static final String USER = "root";
    static final String PASS = "password";

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Enter the Product ID to delete: ");
        int productId = scanner.nextInt();

        String sql = "DELETE FROM Product WHERE productId = ?";

        try (Connection conn = DriverManager.getConnection(DB_URL, USER, PASS);
             PreparedStatement pstmt = conn.prepareStatement(sql)) {

            // Bind the productId parameter
            pstmt.setInt(1, productId);

            // Execute the delete operation
            int rowsDeleted = pstmt.executeUpdate();
            
            // Display appropriate message
            if (rowsDeleted > 0) {
                System.out.println("Product ID " + productId + " was successfully deleted.");
            } else {
                System.out.println("No product found with ID " + productId + ". Avoided deletion.");
            }

        } catch (SQLException e) {
            System.err.println("Database delete failed: " + e.getMessage());
            e.printStackTrace();
        } finally {
            scanner.close();
            System.out.println("Connection and Statement automatically closed.");
        }
    }
}
```
