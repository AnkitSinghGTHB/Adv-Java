# Question 31

**Explain the different states in the Thread Life Cycle in Java.​
At which states can a thread throw an InterruptedException? Justify your answer.
Write a Java program to create a thread that prints "Hello Java" ten times with a delay of
500 milliseconds between each print.**

### 1. Thread Life Cycle States
A Java thread smoothly transitions through various states during its lifecycle, governed by the JVM Thread Scheduler:
1.  **New:** The thread object is instantiated (`new Thread()`), but `start()` has not been called yet.
2.  **Runnable:** The thread is executing or ready for execution, waiting for CPU allocation from the thread scheduler.
3.  **Blocked:** The thread is trying to acquire a monitor lock but cannot because another thread holds it (e.g., waiting to enter a `synchronized` block).
4.  **Waiting (or Timed Waiting):** The thread is temporarily inactive (paused). `sleep()`, `wait()`, or `join()` puts the thread here.
5.  **Terminated:** The thread has completed its `run()` method execution or crashed entirely.

### 2. When Can InterruptedException Occur?
*   A thread can *only* throw an `InterruptedException` when it is in the **Waiting** or **Timed Waiting** state.
*   **Reasoning:** If a thread is sleeping or waiting, it is unresponsive. By calling `.interrupt()` from another thread, the waiting thread immediately wakes up prematurely via the exception, allowing it to gracefully handle the stop or shutdown signal.

### 3. Demonstration Program
```java
class HelloThread extends Thread {
    public void run() {
        for (int i = 1; i <= 10; i++) {
            System.out.println(i + ". Hello Java");
            try {
                // Thread voluntarily moves to Timed Waiting state
                Thread.sleep(500); 
            } catch (InterruptedException e) {
                System.out.println("Thread was interrupted while sleeping.");
            }
        }
    }
}

public class Main {
    public static void main(String[] args) {
        HelloThread ht = new HelloThread();
        ht.start();
    }
}
```
