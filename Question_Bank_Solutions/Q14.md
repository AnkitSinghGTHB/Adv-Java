# Question 14

**Write a Java program to create a GUI application with:
●​ Two text fields (Username and Password)
●​ One button labeled "Login"
●​ One label to display a message
When the button is clicked, check if the username is "admin" and password is "1234".​
Display "Login Successful" or "Invalid Credentials" in the label using event handling.**

---

### Solution

// TODO: Write solution here.

```java
import javax.swing.*;
import java.awt.*;
import java.awt.event.ActionEvent;
import java.awt.event.ActionListener;

public class LoginApp extends JFrame {
    JTextField txtUsername;
    JPasswordField txtPassword;
    JButton btnLogin;
    JLabel lblMessage;

    public LoginApp() {
        setTitle("Login Portal");
        setSize(300, 200);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setLayout(new GridLayout(4, 2, 10, 10)); // Simple grid layout

        // a) Two text fields (Username and Password) 
        txtUsername = new JTextField();
        txtPassword = new JPasswordField(); // Ideal for passwords
        
        // b) One button labeled "Login"
        btnLogin = new JButton("Login");
        
        // c) One label to display a message
        lblMessage = new JLabel("", SwingConstants.CENTER);

        // Grid additions
        add(new JLabel(" Username: "));
        add(txtUsername);
        
        add(new JLabel(" Password: "));
        add(txtPassword);
        
        add(new JLabel("")); // empty cell
        add(btnLogin);
        
        add(new JLabel(" Status: "));
        add(lblMessage);

        // Event Handling
        btnLogin.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                String user = txtUsername.getText();
                String pass = new String(txtPassword.getPassword());

                // Check credentials
                if ("admin".equals(user) && "1234".equals(pass)) {
                    lblMessage.setText("Login Successful");
                    lblMessage.setForeground(Color.GREEN);
                } else {
                    lblMessage.setText("Invalid Credentials");
                    lblMessage.setForeground(Color.RED);
                }
            }
        });
    }

    public static void main(String[] args) {
        new LoginApp().setVisible(true);
    }
}
```

**Sample Output:**
```text
[A GUI Window titled "Login Portal" appears. Entering "admin" and "1234" prints "Login Successful" in green.]
```
